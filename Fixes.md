# (Wifi)

As explained in the [Unsupported List](https://github.com/QuakedK/Service-Destroyer/edit/main/Unsupported%20Features.md#wifi), Wi-Fi is natively unsupported/disabled. However, it can be easily re-enabled by running the [Wi-Fi Fixer](https://github.com/QuakedK/Downloads/blob/main/Turn%20On%20Wifi.bat).
